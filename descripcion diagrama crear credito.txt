La creación de un crédito comienza con el jefe del proyecto a cargo del cual se quiere crear accediendo a los detalles del proveedor del que se va a crear el crédito. Esta información se solicita a la base de datos y se imprime la vista de detalles del proveedor. El usuario pulsa el botón de crear crédito y la interfaz de usuario solicita a la base de datos la lista de proyectos de los que el usuario es jefe, para luego imprimir el formulario de creación de crédito (que incluye un desplegable con los proyectos recibidos). Ya en el formulario, el usuario lo rellena y se envía a la base de datos, la cual comprueba que haya alguna partida en el proyecto que tenga disponible la cantidad requerida. En caso de no haber ninguna, salta un error y finaliza el diagrama, pero si existe alguna, se devuelve la lista de partidas que tienen suficiente y se imprime, para que el usuario pueda seleccionar la partida que quiera, la cual se envía y finalmente se crea el crédito y se recibe una confirmación.
