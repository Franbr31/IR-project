Este diagrama comienza con el usuario entrando a la tabla de proveedores, cual se solicita a la base de datos y se imprime. En la tabla, se selecciona el proveedor del que se quiere pedir, y se reciben los detalles de la base de datos. Ya en la página de detalles, con los datos recibidos se debe contactar con el proveedor para realizar el pedido manualmente (normalmente por teléfono o email) y se recibe una confirmación de pedido. Tras recibirlo, se debe crear el pedido en la base de datos, con lo que se debe acceder a la página de creación de pedido, para finalmente rellenar los datos del formulario, los cuales se envían a la base de datos, creándose el pedido, y se devuelven los detalles de este, finalizando en la página de detalles del pedido creado.
